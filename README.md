# HS Performance PWA

Aplicativo instalável para atletas - desenvolvido com HTML, Firebase e funcionalidades PWA.

## Funcionalidades
- Feedback diário
- Registro de treinos
- Monitoramento de constância
- Questionário RESTQ-36
- Integração com Firebase
- Instalável como app no celular (PWA)
